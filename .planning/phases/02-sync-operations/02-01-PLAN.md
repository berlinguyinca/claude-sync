    **Create src/core/sync-engine.ts** -- The central orchestration module. Imports: `fs` from "node:fs/promises", `path` from "node:path", `scanDirectory` from "./scanner.js", `rewritePathsForRepo`, `expandPathsForLocal` from "./path-rewriter.js", `createBackup` from "./backup.js", and `pushToRemote`, `pullFromRemote`, `fetchRemote`, `getStatus`, `hasRemote`, `addFiles`, `commitFiles` from "../git/repo.js".

    **Types to define and export:**

    ```typescript
    export interface SyncOptions {
      claudeDir: string;
      syncRepoDir: string;
      homeDir?: string; // defaults to path.dirname(claudeDir)
    }

    export interface SyncPushResult {
      filesUpdated: number;
      pushed: boolean;
      message: string;
    }

    export interface SyncPullResult {
      backupDir: string;
      filesApplied: number;
      message: string;
    }

    export interface FileChange {
      path: string;
      type: "modified" | "added" | "deleted";
    }

    export interface SyncStatusResult {
      localModifications: FileChange[];
      remoteDrift: { ahead: number; behind: number };
      excludedCount: number;
      branch: string | null;
      tracking: string | null;
      isClean: boolean;
      hasRemote: boolean;
    }
    ```

    **syncPush(options: SyncOptions): Promise<SyncPushResult>:**
    1. Resolve `homeDir` as `options.homeDir ?? path.dirname(options.claudeDir)`
    2. Check `hasRemote(syncRepoDir)` -- throw `"No remote configured. Add a remote with: git -C <syncRepoDir> remote add origin <url>"` if false
    3. Fetch and check if behind -- `await fetchRemote(syncRepoDir)`, then `const status = await getStatus(syncRepoDir)`. If `status.behind > 0`, throw `"Remote has changes. Run 'claude-sync pull' first."`
    4. Scan `claudeDir` with `scanDirectory` for local files
    5. Copy each file from claudeDir to syncRepoDir. For `settings.json` (check `path.basename(relativePath) === "settings.json"`), apply `rewritePathsForRepo(content, homeDir)`. Use `fs.mkdir(dirname, {recursive: true})` + `fs.writeFile`.
    6. Scan syncRepoDir with `scanDirectory` for repo files. Delete files in repo but not in local scan (using `fs.rm`). This handles user-deleted files.
    7. Get git status. If clean, return `{ filesUpdated: localFiles.length, pushed: false, message: "No changes to push" }`.
    8. Stage with `addFiles(syncRepoDir, ["."])` -- safe because sync repo only has allowlisted files. Commit with message `"sync: update claude config"`. Push with `pushToRemote(syncRepoDir)`.
    9. Return `{ filesUpdated: localFiles.length, pushed: true, message: "Pushed N files to remote" }`.

    **syncPull(options: SyncOptions): Promise<SyncPullResult>:**
    1. Resolve homeDir
    2. Check `hasRemote(syncRepoDir)` -- throw if false (same message as push)
    3. Create backup: `const backupDir = await createBackup(claudeDir, path.join(path.dirname(syncRepoDir), ".claude-sync-backups"))`. Use a `backups` directory alongside the sync repo. If backup throws, propagate the error (never continue without backup).
    4. Pull from remote: `await pullFromRemote(syncRepoDir)`
    5. Scan syncRepoDir with `scanDirectory` for repo files
    6. Copy each file from syncRepoDir to claudeDir. For `settings.json`, apply `expandPathsForLocal(content, homeDir)`. Use `fs.mkdir + fs.writeFile`.
    7. Return `{ backupDir, filesApplied: repoFiles.length, message: "Applied N files from remote. Backup at: <backupDir>" }`.

    **syncStatus(options: SyncOptions): Promise<SyncStatusResult>:**
    1. Resolve homeDir
    2. Check `hasRemote(syncRepoDir)`. If true: `await fetchRemote(syncRepoDir)`, `const gitStatus = await getStatus(syncRepoDir)`. If false: create a mock status with 0 ahead/behind.
    3. Scan claudeDir for local files, scan syncRepoDir for repo files
    4. Compare: for each local file, read content from claudeDir. If `settings.json`, apply `rewritePathsForRepo` to the local content before comparing (to normalize against repo version). Read corresponding repo file. If contents differ: `{ path, type: "modified" }`. If repo file doesn't exist: `{ path, type: "added" }`.
    5. For each repo file not in local scan: `{ path, type: "deleted" }`.
    6. Count excluded: total files in claudeDir (recursive readdir, filter isFile) minus allowlisted count.
    7. Return structured result.
